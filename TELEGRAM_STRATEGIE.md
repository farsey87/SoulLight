## 2. TELEGRAM-BOT FUNKTIONEN

### 🤖 „SoulLight Bot" – Automatisierung

#### A) Buchungssystem
```
Ablauf:
1. User schreibt /buchen oder klickt "Termin buchen"
2. Bot zeigt Kategorien: Hypnose | Karma | Reinkarnation | Spirit Healing
3. User wählt Kategorie
4. Bot zeigt verfügbare Zeitslots (verknüpft mit Calendly/Cal.com)
5. User wählt Termin
6. Bot bestätigt & sendet Zoom-Link
7. 24h vorher: Automatische Erinnerung
8. 1h vorher: Zoom-Link nochmal + Vorbereitungstipps
```

#### B) FAQ-Bot
```
Befehle:
/start        → Willkommensnachricht + Menü
/angebote     → Übersicht aller Dienstleistungen & Preise
/pakete       → Premium-Pakete anzeigen
/buchen       → Buchungsprozess starten
/telegram     → Infos zu den Gruppen (Free/Premium/VIP)
/faq          → Häufig gestellte Fragen
/kontakt      → Kontaktmöglichkeiten anzeigen
/meditation   → Tägliche kostenlose Meditation senden
/affirmation  → Zufällige Affirmation senden
```

#### C) Automatische Nachrichten
| Trigger | Nachricht |
|---------|-----------|
| Neuer User startet Bot | Willkommen + Angebots-Übersicht + Link zur kostenlosen Gruppe |
| 24h vor Session | Erinnerung + Vorbereitungstipps + Zoom-Link |
| 1h vor Session | Zoom-Link + „Finde einen ruhigen Ort" |
| Nach Session (24h) | Feedback-Anfrage + Nachsorge-Tipps |
| 7 Tage nach Session | Check-in + Angebot für Folge-Session |
| Geburtstag | Persönliche Glückwünsche + kleines Geschenk (z.B. Meditation) |

#### D) Payment-Integration
- **Telegram Stars** oder **externe Zahlung** (Stripe/PayPal-Links)
- Bot kann Links zu Zahlungsseiten senden
- Nach Zahlung: Automatischer Zugang zur Premium-Gruppe

## 5. TECHNISCHE UMSETZUNG

### Benötigte Tools
| Tool | Zweck | Kosten |
|------|-------|--------|
| **Telegram Bot API** | Bot-Erstellung | Kostenlos |
| **Python + python-telegram-bot** | Bot-Programmierung | Kostenlos |
| **Cal.com / Calendly** | Terminbuchung | Free / ab 8€/Monat |
| **Stripe** | Zahlungsabwicklung | 1,4% + 0,25€ pro Transaktion |
| **Google Sheets** | Kundendaten & Tracking | Kostenlos |
| **Make.com / Zapier** | Automatisierungen | Free / ab 19€/Monat |
| **Canva** | Grafiken für Posts | Free / ab 12€/Monat |

### Bot-Architektur
```
Telegram Bot
    ├── /start → Willkommen + Hauptmenü
    ├── /buchen → Inline-Keyboard mit Kategorien
    │       ├── Hypnose → Zeitslots (Cal.com API)
    │       ├── Karma → Zeitslots
    │       ├── Reinkarnation → Zeitslots
    │       └── Spirit Healing → Zeitslots
    ├── /angebote → Inline-Keyboard mit Details
    ├── /premium → Infos + Zahlungslink
    ├── /meditation → Random Meditation senden
    ├── /affirmation → Random Affirmation
    └── Scheduled Tasks
            ├── Tägliche Affirmation (07:00)
            ├── Session-Erinnerungen (24h/1h vorher)
            └── Follow-up Messages (nach Sessions)
```
